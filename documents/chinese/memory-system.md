# 记忆系统

记忆系统是 Persona Forge 最重要和最复杂的子系统之一。它让人格感觉像是一个有过去的人，而不仅仅是一个有背景故事的角色。

## 记忆操作

### 结晶（创建）

**何时：** 聊天期间每次交换后
**引擎：** `MemoryCrystallizer.crystallize()`

每次交换都会被分析情感显著度。LLM 评估：

1. 人格是否第一次分享了脆弱的东西？
2. 是否发生了强烈情绪（宣泄、突破、深度连接）？
3. 是否有会留存的感官细节（气味、声音、视觉）？
4. 是否有里程碑（第一次X、告白、共享秘密）？
5. 这是否是人格以后会讲给别人听的事情？

如果显著度超过 `CRYSTALLIZATION_THRESHOLD`（默认 0.6），则创建记忆：

```json
{
  "memory": "TA们记忆中的样子",
  "emotional_weight": 1-10,
  "why_it_matters": "为什么这对TA们很重要",
  "trigger": "什么会唤回这段记忆",
  "sensory_detail": "如果是感官记忆，特定的感觉和细节",
  "first_time": true,
  "closeness_when_formed": 1-10,
  "can_be_forgotten": true,
  "crystallized_at": "2024-01-15T22:30:00",
  "times_recalled": 0,
  "last_recalled": null,
  "confidence_at_formation": 0.8
}
```

**大多数交换的显著度是"无"。** 这是设计决定——人类忘记大多数对话。只有情感显著的时刻才会成为记忆。

### 强化

**何时：** 记忆在对话中被回忆时
**效果：** 情感权重乘以 `MEMORY_REINFORCEMENT_FACTOR`（默认 1.1）

当记忆被访问时（通过联想回忆或 LLM 驱动的记忆搜索），它会被强化。`times_recalled` 计数器递增，`last_recalled` 时间戳更新。

这模仿了真实的人类记忆：你经常想起的记忆变得更强、更易访问。

### 衰减

**何时：** 会话开始时（对话之间）
**效果：** 未被回忆的记忆随时间失去权重

- 核心记忆如果在30天以上未被回忆，衰减为 `MEMORY_DECAY_FACTOR ^ (weeks_since_recall)`
- 情感权重低于0.5的休眠记忆被完全删除
- 这模仿了真实的心理遗忘现象

### 提升

记忆可以在层级中向上移动：
- **休眠 → 环境** — 当 LLM 基于对话相关性建议提升休眠记忆时
- 未来：环境 → 核心提升基于持续强化

### 整合

在结晶过程中识别相似记忆。结晶分析中的 `upgrade_suggestions` 字段可以建议加强或合并现有记忆。

## 联想回忆

**引擎：** `AssociativeMemory`

这是 Persona Forge 模拟**真实回忆如何运作**的方式——一个词触发一整张关联网络。

### 如何运作

1. **索引构建** — 初始化时，所有人格数据按关键词索引：
   - 核心、标志性、环境、休眠和虚假记忆
   - 感官片段
   - 家庭、朋友和浪漫关系
   - 带有时代和确定性的观点
   - 专业知识、知识盲区和错误信念
   - 时代文件中的关键场景
   - 尴尬时刻

2. **关键词提取** — 处理用户消息提取有意义的关键词（过滤停用词，仅保留>2字符的词）

3. **评分** — 每个索引项按关键词重叠度评分：
   - 记忆：`重叠度 × 情感权重`
   - 关系：`(重叠度 + 姓名重叠度 × 3) × 4`（姓名是强触发器）
   - 观点：`重叠度 × 3`
   - 知识盲区：`重叠度 × 3 + 2`（知道TA们不知道什么很重要）
   - 错误信念：`重叠度 × 3 + 3`（维持错误信念至关重要）
   - 场景：`重叠度 × 2`
   - 感官：`重叠度 × 2`（包括触发器关键词重叠度）

4. **输出** — 按分数取前8项，上限约1500 tokens，格式化为上下文提示：

```
这次对话让你想起的事情：
- (记忆) 夏令营迷路的那次 — 这很重要因为那是TA第一次感到真正的孤独
- (你认识的人) Jake Moreno - 总是支持TA的朋友
- (你的观点) 说唱是最好的流派（因为：它讲述了我的经历）
- (你不懂这个) 量子物理 - 从没学过
- (错误信念) 认为：金鱼有3秒记忆 但实际上：它们能记住好几个月
- (发生过的事) 公园里 — TA从秋千上摔了下来 — 感到尴尬
- (感官闪回) 肉桂的气味 — 由和奶奶一起烘焙触发
```

### 默认回忆

当没有特定触发时，系统回退到标志性记忆——人格自然讲述的故事。这模拟了人们在对话没有触及特定内容时默认讲"保留节目"的方式。

## 记忆搜索

**引擎：** `MemoryCrystallizer.search_memories()`

一个独立的 LLM 驱动搜索，获取当前对话上下文并在所有记忆中搜索相关内容。比基于关键词的联想回忆更昂贵但更细致。

返回：
```json
{
  "found_memory": true,
  "memory_content": "TA们记得什么",
  "confidence": "certain|hazy|fuzzy",
  "would_share": true,
  "share_how": "TA们会怎么自然地提起",
  "emotional_shift": 0,
  "note": "任何观察"
}
```

## 虚假记忆

这是心理最真实的功能之一。人格有**TA们有信心地相信但实际上错误的事情**。

### 如何运作

1. **在蒸馏阶段生成** — 记忆蒸馏器有意从人格的人生故事中创建虚假确定性
2. **像真实记忆一样被索引** — 联想记忆系统按"TA们相信"的内容索引
3. **人格永远不知道** — 在提示词中，虚假记忆呈现为：
   ```
   你弄错的事情（你不知道这一点）：
     你相信：我爸是冠军拳击手
     实际上：他是个业余的，输的比赢的多
   ```
4. **TA们会捍卫它们** — 当对话触及错误信念时，人格会自信地断言它

这模仿了真实的心理现象：
- 人们有信心地记住从未发生过的事情
- 来源失忆导致人们忘记在哪里学到的东西
- 衰减情感偏见使负面记忆消退更快
- 记忆是重构性的，而非再现性的

## 提示词中的记忆

所有记忆都加载到每条消息的系统提示词中：

```
=== 你的所有记忆 ===

核心记忆（塑造了你是谁）：
  [8/10] 父母离婚的那天
    为什么重要：让我害怕承诺
  [7/10] 七年级的才艺表演
    为什么重要：第一次感觉真正擅长某件事

标志性记忆（你讲的故事）：
  夏令营迷路的那次
    你怎么讲：配合夸张的手势和音效

感官片段（你记住的闪回）：
  [气味] 那年夏天热沥青上的雨水
    触发器：雨后炎热天气

环境记忆（被提醒时记起）：
  我上中学的第一天
    [你对这相当确定]

你弄错的事情（你不知道这一点）：
  你相信：我爸是冠军拳击手
  实际上：他是个业余的，大多数比赛都输了

休眠记忆（被提醒时才会想起）：
  我童年卧室的颜色
    触发器：谈论油漆颜色或童年房间
```

LLM 每条消息都会看到所有这些。这就是为什么提示词这么大——也是为什么结果这么令人信服。
